Please use the following format to get the data using the GET Request

https://heroku-app-us-admissions.herokuapp.com/api/<GRE>/<GPA>/<RANK>

In place of <GRE> enter something like 100 and so on in each of the 3 values within range

Allowed Range

0<=GRE<=800

0<=GPA<=4

1<=RANK<=4

Eg. type "https://heroku-app-us-admissions.herokuapp.com/api/100/2/4" in url box or click to see this example.

Eg. type "https://heroku-app-us-admissions.herokuapp.com/api/650/4/2" in url box or click to see this example.

If return value 0 then not admitted else if 1 then admitted

If you want to do the POST request, please follow the following format:

The POST request has to be sent at URL: https://heroku-app-us-admissions.herokuapp.com/api

The body should contain the data in the JSON format :

{ "gre":<int:gre>, "gpa":<float:gpa>, "rank":<int:rank> }

Eg.: { "gre":650, "gpa":4, "rank":2 }

If you want to do PUT request to update the data please follow the following format:

The PUT request has to be sent at URL: https://heroku-app-us-admissions.herokuapp.com/api

The body should contain the data in the JSON format :

{ "id":<int:id> "gre":<int:gre>, "gpa":<float:gpa>, "rank":<int:rank> }

NOTE: In GRE, GPA, RANK any 2 are optional but minimum of 1 parameter is required.

If you want to do DELETE request to delete the data please follow the following format:

The PUT request has to be sent at URL: https://heroku-app-us-admissions.herokuapp.com/api

The body should contain the data in the JSON format :

{ "id":<int:id> }

NOTE: ID is a required parameter.
